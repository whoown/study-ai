请帮写一个python 脚本，便于在服务器上一键配置 sing-box + vless + reality 方案的科学上网环境。具体要求如下：

1. 脚本使用 python 编写，具备充分的中文注释；

2. 所有需要人工配置的参数均放在脚本头部，便于人工填写；

3. 脚本需要具备幂等性，即多次调用不影响结果，不会给服务器带来负面影响；

4. SNI 作为一个可配置项，默认值是 loewe.com。

5. 流量出口需要支持至少三种（后续可能增减），包括：

5.1 ：客户端直连当前节点；

5.2 ：当前节点作为中转节点，将请求再转发到某个美国 vps 上去。

5.3 当前节点作为中转节点，将请求再转发一层美国的家宽 IP 代理（需支持 http 和 socks5）

6. 这三种流量出口，采用不同的 UUID 区分，其余端口等配置完全一样。

7. 最终输出三个适配 V2rayN/shadowRocket 等客户端的 URL。用户可以手动切换三种路线。

8. 注意本服务器环境内存/CPU/磁盘空间都比较有限，所以注意开启包括 BBR 、xtls-rprx-vision 在内的各种优化项。
